CSS3D for the iPad

Why CSS3D?

Apple Don't allow WebGL.

Canvas is still slow.

SVG support is good but is an unfriendly syntax.

CSS3D elements are hardware accelerated and easily styled.

Webkit Transforms

Syntax in CSS

section{-webkit-transform:rotateY(45deg);}

In JavaScript

document.querySelector("section").style.webkitTransform = "rotateY(45deg)";

Touch Events

Event Listeners

document.querySelector("div").addEventListener("touchstart", touchStart, false);

Event Handlers

touchStart = function(event){ event.preventDefault(); console.log("started x = "+ event.touches[0].pageX); console.log("started y = "+ event.touches[0].pageY); };
